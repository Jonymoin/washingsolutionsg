# WashingSolutionsSG Website

Singapore washing machine repair service website built with React + Vite + Tailwind CSS.

## Quick Start

```bash
npm install
npm run dev
```

Then open http://localhost:5173 in your browser.

## Build for production

```bash
npm run build
npm run preview
```

## Tech Stack

- React 18
- Vite 6
- Tailwind CSS v4
- Lucide React (icons)
- TypeScript
